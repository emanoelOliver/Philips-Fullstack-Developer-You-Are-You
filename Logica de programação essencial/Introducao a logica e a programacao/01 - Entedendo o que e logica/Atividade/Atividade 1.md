# Atividade #1

## Crie um mapa mental para resolver um determinado problema, pro exemplo, calcular a média aritmética de 4 notas, sabendo que as notas são as seguintes:

1. Nota : 5
2. Nota : 7
3. Nota : 10
4. Nota : 3 



|      Notas      |             **Média**             |
| :-------------: | :-------------------------------: |
| **Nota 1** : 5  |                 5                 |
| **Nota 2** : 7  |                 7                 |
| **Nota 3** : 10 |                10                 |
| **Nota 4** : 3  |                 3                 |
|                 | **Resultado** : **25 / 4 = 6,25** |
